Detailed Description

Menu que apresenta as opções do programa.

Essa classe é iniciada junto com o programa e mostra ao usuário as opções do sistema (dados, operações ou resultados). As ações escolhidas são interpretadas no controlador e a saída apropriada é gerada. Faz herança com a classe Menu.
